모각코 2회

  • Mogaco
  • Date : 2020/01/16

문자열은 "" 이나 ''안에 넣어준다.
""안에 "를 넣으려면 "앞에 \넣어서 \"(escape)로 만들어주면 입력이 된다.
typeof 인자 를 넣으면 인자의 타입(String or number)를 알려준다.

\n : 줄바꿈
문자열.length : 문자열의 길이를 알려줌
문자열.indexOf("문자") : 해당문자가 위치한 인덱스를 알려줌

변수란 값을 담는 그릇이다. 숫자 뿐만 아니라 문자도 가능하다. 또한 불린도 가능
var 변수명 = 값

== : 동등 연산자, 값이 같으면 true 다르면 false를 반환된다. java와 다르게 문자열도 비교 가능
=== : 일치 연산자, 값이 정확하게 같으면 true 다르면 false가 반환된다.
1 == '1' 은 true를 반환 1 === '1' 은 false를 반환한다.
undefined는 값이 없는 것이고 null은 값이 프로그래머에 의해 의도적으로 값이 없는 것이다.
동등 연산자에서 true와 비교할 때 1은 true 0은 false로 간주한다.
예외적인 부분을 잘 알고 있지 않은 이상 ===을 사용하자

!는 부정을 의미한다.
!= : 같지 않다는 의미로 동등연산자의 정반대의 결과를 보여준다.
!== : 정확하게 같지 않다는 의미이다.

promprt('인자') : 인자가 화면에 뜨면서 사용자로 부터 값을 받을 수 있다.
boolean의 대체제 : 숫자 0은 false 숫자 1은 true로 간주한다.''는 false '내용'은 true로 간주한다. undefined도 false라 간주한다.
