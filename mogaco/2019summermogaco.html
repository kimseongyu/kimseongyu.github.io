HTML & CSS라는 책의 1장부터 5장까지 HTML을 공부할 예정이다.

모각코 1회

  • Mogaco
  • Date : 2019/07/09


1장 구조
html : HTML 코드를 나타낸다.
body : body 안에 있는 모든 콘텐츠는 메인 브라우저 창에 표시
head : 페이지에 대한 정보가 포함되며 body 앞에 쓰인다.
title : 탭에 표시되는 글씨


2장 텍스트 h1 : 메인제목
h2 ~ h6 : n레벨 제목
p : 단락
b : 굵은 글씨
i : 기울인 글씨
sup : 위침자
sub : 아래침자
br : 줄바꿈
hr : 가로줌


3장 목록
ol : 순서가 있는 목록
ul : 순서가 없는 목록
li : 목록들은 li안에 배치


4장 링크
형식 : (a href="주소" ~~)텍스트(/a)
a : href속성의 값으로 이동하는 링크를 만듬
ㄴ절대url : 다른 사이트에 대한 링크인 경우
ㄴ상대url : 동일 사이트에 대한 링크인 경우


5장 이미지
형식 : (img src="주소" ~~)
img : 이미지를 추가할 때 사용하는 요소, 닫기 태그가 없다.
ㄴsrc : 블라우저에게 이미지파일의 경로를 알려줌
ㄴalt : 이미지를 볼 수 없을때 설명을 제공
ㄴtitle : 마우스를 이미지에 올렸을 때 설명 제공
ㄴheight, width : 사진의 크기 지정

모각코 2회 목표

  • Mogaco
  • Date : 2019/07/16

HTML & CSS라는 책의 6장부터 9장까지 HTML을 공부할 예정이다.

모각코 2회

  • Mogaco
  • Date : 2019/07/16


6장 테이블
table:테이블을 생성하는데 사용
tr:각 행을 작성하는데 사용
td:행 각각의 셀을 작성하는데 사용
th:td와 같은 방식이지만 행이나 열의 머리말을 나타내는데 사용
ㄴcolspan:병합해야 하는 열의 개수를 지정
ㄴrowspan:병합해야 하는 행의 개수를 지정


7장 폼
form:폼 컨트롤은 내부에 위치
ㄴaction:정보를 수신할 서버페이지의 URL
ㄴmethod:정보를 보내는 방법
input:다른 폼 컨트롤을 만드는데 사용
ㄴtext:한줄의 텍스트박스 생성
ㄴpassword:마스크처리 되는 텍스트박스 생성
ㄴradio:라디오 버튼을 생성
ㄴcheckbox:체크박스를 생성
ㄴfile:전송할 파일을 선택할 수 있도록 함
button:버튼의 표시방법을 제어
fieldset:관련된 폼컨트롤을 그룹화 할 수 있음


8장 추가마크업
!DOCTYPE html:HTML의 버전을 알려줌
!-- --: 주석 처리
ㄴid:다른요소들과 구분할 때 사용
ㄴclass:몇 개의 특정요소를 식별할때 사용
div:요소들을 하나의 블록 레벨박스로 그룹화
span:div의 인라인 형태이다.
iframe:페이지에서 작은 창을 통해 다른 페이지를 여는 것
meta:페이지에 대한 정보를 가짐


9장 플래시, 오디오, 비디오
video:비디오 재생을 제어
ㄴsrc:비디오에 대한 경로
ㄴposter:비디오를 내려받거나 재생하기 전 표시할 이미지
ㄴwidth, height:플레이어의 크기를 지정
ㄴcontrols:브라우저가 컨트롤을 제공해야 함을 나타냄
ㄴautopalys:파일이 자동으로 재생되도록 지정
source:재생할 파일의 위치를 지정
audio:오디오 재생을 제어
ㄴsrc:오디오에 대한 경로지정
ㄴcontrols:컨트롤을 표시할지에 대한 여부
ㄴautoplay:오디오가 자동으로 재생을 시작함을 나타냄

HTML & CSS라는 책의 10장부터 14장까지 CSS을 공부할 예정이다.

모각코 3회

  • Mogaco
  • Date : 2019/07/18


10장 CSS소개
형식:선택자{프로터피:값;}
선택자:규칙을 적용할 요소를 나타낸다
프로퍼티:변경하려는 요소의 부분
값:프로퍼티에 대해 사용하려는 설정을 지정

link:CSS파일의 위치를 브라우저에게 알려주기 위해 HTML문서에 사용
ㄴhref:CSS파일의 경로를 가리킨
ㄴtype:연결할 문서의 타입을 지정, CSS의 경우 "test/css"
ㄴrel:HTML파일과의 관계를 지정, CSS파일과 연결할때 값이 "stylesheet"여야 한다.
style:CSS규칙을 HTML페이지 내에 포함시킬 수 있음, type속성 사용
상속:CSS규칙은 자식요소에 상속된다.


11장 색상
color:요소 내부에 있는 텍스트의 색상을 지정
ㄴRGB값:빨,녹,파의 관점으로 표현 ex) rgb(00,100,90)
ㄴ헥사코드:빨,녹,파의 양을 헥사코드로 표현 ex) #ee3e80
ㄴ색상명:미리 정의된 147개의 색상명으로 표현 ex) gray
background-color:해당 박스에 대한 배경색을 설정
ㄴrgba:RGB값+투명도 ex) rgba(00,00,00,0.5)
ㄴhsl:색조,채도,밝기 표현 ex) hsl(0,0%,80%)
ㄴhsla:hsl+투명도 ex) hsla(0,0%,80%,0.5)
opacity:투명도 표현, 투명도 표현 0~1사이의 숫자로 표시 ex)0.5=50%


12장 텍스트
font-family:사용할 서체를 지정
font-size:폰트의 크기를 지정, 픽셀,퍼센트,em중 하나를 단위로 함
@font-face:폰트를 내려받아 사용가능
text-transform:텍스트를 대소문자로 변경
ㄴuppercase:텍스트를 모두 대문자로 나타냄
ㄴlowercase:텍스트를 모두 소문자로 나타냄
ㄴcapitalize:각 단어의 첫 글자를 대문자로 나타냄
text-decoration:텍스트에 밑줄이나 취소선 지정
ㄴunderline:텍스트에 밑줄을 추가
ㄴoverline:텍스트에 윗줄을 추가
ㄴline-through:텍스트 가운데에 줄을 긋는다
ㄴblink:텍스트를 깜빡이게 함
line-height:텍스트 사이의 행간 크기조정
letter-spacing, word-spacing: 문자, 단어간격
text-align:텍스트의 정렬을 설정
ㄴleft:텍스트를 왼쪽으로 정렬
ㄴright:텍스트를 오른쪽으로 정렬
ㄴcenter:텍스트를 가운데로 정렬


13장 박스
width, height:박스의 크기를 지정
overflow:박스의 크기보다 내용이 많을 때 사용
ㄴhidden:박스보다 큰 부분의 콘텐츠를 숨김
ㄴscroll:박스에 스크롤바를 추가해 콘텐츠를 볼 수 있게 함
border-width:테두리의 두께를 px단위로 조정
border-style:테두리 스타일 지정
border-color:테두리 색상 지정
border:너비, 스타일, 색상 순서대로 지정
border-image:이미지 테두리
border-shadow:박스 그림자
border-radius:둥근 모서리, 값은 반지름의 크기를 의미


14장 목록, 테이블, 폼
목록
list-style-type:글머리 기호 스타일을 지정
list-style-image:글머리 기호 이미지
list-style-position:글머리 기호 위치
list-style:type, position, image 값을 한꺼번에 지정
테이블
empty-cells:테두리 표시여부를 결정
border-spacing:인접한 셀 사이의 간격을 조정
border-collapse:셀사이의 거리 여부
폼컨트롤 정렬하기
1.float으로 제목을 컨트롤 왼쪽에 배치
2.width로 제목 너비를 설정해 폼이 같은 열에 배치
3.text-algin으로 제목을 오른쪽으로 정렬
