HTML & CSS라는 책의 1장부터 5장까지 HTML을 공부할 예정이다.

모각코 1회

  • Mogaco
  • Date : 2019/07/09


1장 구조
html : HTML 코드를 나타낸다.
body : body 안에 있는 모든 콘텐츠는 메인 브라우저 창에 표시
head : 페이지에 대한 정보가 포함되며 body 앞에 쓰인다.
title : 탭에 표시되는 글씨


2장 텍스트 h1 : 메인제목
h2 ~ h6 : n레벨 제목
p : 단락
b : 굵은 글씨
i : 기울인 글씨
sup : 위침자
sub : 아래침자
br : 줄바꿈
hr : 가로줌


3장 목록
ol : 순서가 있는 목록
ul : 순서가 없는 목록
li : 목록들은 li안에 배치


4장 링크
형식 : (a href="주소" ~~)텍스트(/a)
a : href속성의 값으로 이동하는 링크를 만듬
절대url : 다른 사이트에 대한 링크인 경우
상대url : 동일 사이트에 대한 링크인 경우


5장 이미지
형식 : (img src="주소" ~~)
img : 이미지를 추가할 때 사용하는 요소, 닫기 태그가 없다.
src : 블라우저에게 이미지파일의 경로를 알려줌
alt : 이미지를 볼 수 없을때 설명을 제공
title : 마우스를 이미지에 올렸을 때 설명 제공
height, width : 사진의 크기 지정
