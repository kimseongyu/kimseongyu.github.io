#include <stdio.h>

char name[50];

int main(){
	setvbuf(stdout,0,2,0);
	printf("Name:");
	read(0,name,50);
	char buf[20];
	printf("Try your best:");
	gets(buf);
	return ;
}

ret2sc main함수의 assembly코드이다.

main+102에 브레이크를 걸고 실행시켜 보았다.

name의 주소가 0x56557040이라고 뜬다. 다른 문제풀이랑은 주소가 다른데 잘못 된 것 같은 느낌이 든다.
