Markdown

Markdown이란?

일반 텍스트 문서의 양식을 편집하는 문법이다. README 파일이나 온라인 문서, 혹은 일반 텍스트 편집기로 문서 양식을 편집할 때 쓰인다. 마크다운을 이용해 작성된 문서는 쉽게 HTML 등 다른 문서형태로 변환이 가능하다.

## h2
### h3
#### h4
##### h5
###### h6

    h1

    h2

    h3

    h4

    h5
    h6

  • 글씨형태

    *기울인 글씨*
**굵은 글씨**
***굵고 기울인 글씨***
<u>밑줄있는 글씨</u>
~~취소선있는 글씨~~

    기울인 글씨 굵은 글씨 굵고 기울인 글씨 밑줄있는 글씨 취소선있는 글씨
