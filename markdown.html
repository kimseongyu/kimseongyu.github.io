데이터베이스 품질과 표준화

데이터 품질관리 프레임워크

 

데이터 값

  • 표준 데이터 : 공통된 형식과 내용으로 정의하여 사용하는 표준 관련 데이터
  • 모델 데이터 : 데이터 모델을 운용, 관리하는데 필요한 데이터
  • 관리 데이터 : DB를 효과적으로 운영, 관리하는데 필요한 데이터
  • 업무 데이터 : 기관이나 기업의 업무 및 비즈니스를 수행하는데 필요한 데이터

 

데이터 구조

  • 개념 데이터 모델 : 업무 요건을 충족하는 데이터의 주제영역, 핵심 엔터티, 핵심 관계를 정의한 모델
  • 데이터 참조 모델 : 업무 영역별, 주제 영역별로 표준 데이터 집합과 관리 항목들을 정의한 모델
  • 논리 데이터 모델 : 논리적인 데이터 집합, 관리 항목, 관계를 정의한 모델
  • 물리 데이터 모델 : DBMS의 특성과 성능을 고려하여 논리 데이터 모델을 구체화시킨 모델
  • 데이터베이스 : 물리 모델을 구현한 결과물이며 구축된 실제 데이터가 저장되는 데이터 저장소
  • 사용자 뷰 : 데이터를 제공하는 정보시스템상의 화면이나 출력물

 

데이터 관리 프로세스

  • 데이터 표준 관리 : 데이터를 정제하고 개선하는 활동
  • 요구사항 관리 : 사용자의 요구를 수집하고 분류하여 반영하는 작업
  • 데이터 모델 관리 : 데이터 모델을 유지하는 작업
  • 데이터 흐름 관리 : DB에 저장하고 가공하는 과정을 관리하는 작업
  • DB 관리 : DB를 안정적으로 운영, 관리하는데 필요한 작업
  • DB 보안 관리 : 허가되지 않은 일련의 행위로부터 안전하게 보호하는 작업
  • 데이터 활용 관리 : 품질을 측정하고 품질 미충족의 원인을 분석하는 작업

 

데이터 품질관리 성숙모형

1단계에서부터 5단계로 정의됨, 성숙수준이 높을수록 체계적이고 정교한 관리가 수행됨을 의미함

 

데이터 표준화

데이터 정보 요소에 대해 명칭, 정의, 형식, 규칙에 대한 원칙을 수립하여 이를 전사적으로 적용하는 것

 

데이터 표준 정의

  • 표준 단어의 도출 전체 개요 : 모든 용어를 단어 단위로 분할하여 도출함
  • 단어 분할 : 수집된 용어를 업무상 사용되며 일정한 의미를 갖고 있는 최소 단위의 단어로 분할
  • 이음동의어 처리 : 의미가 동일한 단어들에 대해 하나의 대표 단어를 표준으로 선정함
  • 표준 도메인 사전 도출 : 각 속성은 임의의 도메인에 할당되어야 하며, 하나 이상의 도메인에 복수로 할당될 수 없음
  • 표준 코드 사전 도출 : 사용되고 있는 모든 코드들을 수집하여 동일 코드를 파악하고 통합하여 표준 코드를 정의함
  • 표준 용어 사전 도출 : 추출된 표준 단어를 조합하여 표준 용어를 생성함
