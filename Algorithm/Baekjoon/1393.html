1393번 음하철도 구구팔

  • Algorithm
  • Java
  • Date : 2020/07/03
  • https://www.acmicpc.net/problem/1393

문제요약
달리고 있는 열차에서 철도가 정류장에 가장 많이 근접했을 때 뛰어내리고자 한다.
어디서 뛰어내려야 하는가? 

                            
public class Baekjoon1393 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        int sx = sc.nextInt();
        int sy = sc.nextInt();
        int cx = sc.nextInt();
        int cy = sc.nextInt();
        int movex = sc.nextInt();
        int movey = sc.nextInt();
        int movemax = Math.max(movex, movey);
        int movemin = Math.min(movex, movey);
        while(movemin > 0) {	// 유클리드 호제법
            int tmp = movemax;
            movemax = movemin;
            movemin = tmp % movemin;
        }
        movex = movex / movemax;
        movey = movey / movemax;
        int minx = 10000;
        int miny = 10000;
        while(true) {
            if((sx - cx)*(sx - cx) + (sy - cy)*(sy - cy) >= (sx - minx)*(sx - minx) + (sy - miny)*(sy - miny)) {
                break;
            }else {
                minx = cx;
                miny = cy;
            }
            cx += movex;
            cy += movey;
        }
        System.out.print(minx + " " + miny);
    }
}

유클리드 호제법을 사용하여 이동하는 좌표들의 최대 공약수를 구하여 최소로 이동하는 좌표위치를 구한다. 현재 위치를 최소 이동횟수로 움직이면서 가장 거리가 짧은 좌표를 구한다. 가장 거리가 짧은 좌표 이후에는 다시 거리가 멀어지기 때문에 반복문을 멈춰준다.
