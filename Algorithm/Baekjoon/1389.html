1389번 케빈 베이컨의 6단계 법칙

  • Algorithm
  • Date : 2020/01/17
  • https://www.acmicpc.net/problem/1389

문제요약
케빈 베이컨 게임은 임의의 두 사람이 최소 몇 단계 만에 이어질 수 있는지 계산하는 게임이다.
BOJ 유저의 수와 친구 관계가 입력으로 주어졌을 때, 케빈 베이컨의 수가 가장 작은 사람을 구하는 프로그램을 작성하시오. 

                            
public class Baekjoon1389 {
    private static int user;
    private static int relationNum;
    private static int[] bacon;
    private static int[][] bfs;
    private static boolean[] visit;

    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        int smallNum = 1;
        
        user = input.nextInt();
        relationNum = input.nextInt();
        bacon = new int[user];
        bfs = new int[user + 1][user + 1];
        visit = new boolean[user + 1];
        for (int i = 0; i < relationNum; i++) {
            int x = input.nextInt();
            int y = input.nextInt();
            bfs[x][y] = bfs[y][x] = 1;
        }
        for (int i = 1; i <= user; i++) {
            bacon[i-1] = baconFunc(i);
            visit = new boolean[user + 1];
        }
        for(int i = 1; i < user; i++) {
            if(bacon[i] < bacon[i-1]) {
                smallNum = i+1;
            }
        }
        System.out.println(smallNum);
    }

    public static int baconFunc(int i) {
        Queue q = new LinkedList();
        int[] size = new int[2];
        size[0] = 1;
        int pluseNum = 1;
        int bacon = 0;
        
        q.offer(i);
        visit[i] = true;

        while (!q.isEmpty()) {
            int temp = q.poll();
            for (int j = 1; j <= user; j++) {
                if (bfs[temp][j] == 1 && visit[j] == false) {
                    q.offer(j);
                    visit[j] = true;
                    bacon += pluseNum;
                    size[1]++;
                }
            }
            if(--size[0] == 0) {
                size[0] = size[1];
                size[1] = 0;
                pluseNum++;
            }
        }
        return bacon;
    }
}

bfs로 풀고 있는데 30%에서 막힌다. 머릿속에선 완벽한데 왜인지 모르겠다. ㅠㅠ
기막힌 방법이 떠오르면 다시 풀어봐야겠다.

bfs(너비 우선 탐색)란?
루트 노드(혹은 다른 임의의 노드)에서 시작해서 인접한 노드를 먼저 탐색하는 방법
