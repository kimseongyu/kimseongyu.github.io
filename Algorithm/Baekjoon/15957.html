15957번 음악 추천

  • Algorithm
  • Java
  • Date : 2020/02/12
  • https://www.acmicpc.net/problem/15957

문제요약
음악 간 유사도를 트리의 형태로 만든 뒤 사용자의 입력에 따라 추천할 음악을 결정하는 알고리즘을 만들려고 한다.
이 알고리즘의 가중치에는 노드의 수로 나누어떨어지지 않는 경우 몫만이 분배된다.
가수의 평균 점수는 해당 가수의 노래에 나눠진 가중치의 총합에 해당 가수의 노래의 수를 나누어서 얻을 수 있다.
곡의 수 N, 추천 알고리즘의 결과 데이터의 수 K, 목표 점수 J가 주어진다.
2번째 줄에는 N-1개의 곡 번호가 주어지는데, 이는 2번 곡부터 해당 곡의 부모 노드가 되는 곡의 번호이다.
3번째 줄에는 1번 곡부터 해당 곡을 부른 가수의 번호 N개가 주어진다.
다음 K개의 줄에 추천 알고리즘의 결과 데이터가 하나씩 주어진다.
각 줄의 처음에는 데이터가 계산된 시간, 다음에는 점수가 부여되는 서브트리의 루트가 되는 곡의 번호, 마지막에는 서브트리에 부여할 가중치가 주어진다.
출력은 1번 곡부터 해당 곡을 부른 가수의 평균 점수가 J를 넘게 되는 시간을 출력한다.
점수가 J를 넘는 일이 없는 경우 -1을 출력한다.
