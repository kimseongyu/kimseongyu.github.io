5766번 할아버지는 유명해!

  • Algorithm
  • Java
  • Date : 2020/02/04
  • https://www.acmicpc.net/problem/5766

문제요약
랭킹에 선수의 이름이 오를 때마다 선수의 포인트가 1포인트씩 오른다고 하고
N주 동안의 매주 상위 M명의 랭킹 정보가 주어질 때
2등인 선수를 출력하시오. 두 명 이상인 경우오름차순으로 출력하시오.
N과 M이 모두 0으로 주어질때 입력이 끝난다. 

                            
public class Baekjoon5766 {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        
        while(true) {
            int week = input.nextInt();
            int ranker = input.nextInt();
            int[] people = new int[10000];
            if(week == 0)break;
            for(int i = 0; i < week * ranker; i++)
                people[input.nextInt()-1]++;
            int max = people[0];
            for(int i = 1; i < 10000; i++)
                if(max < people[i])
                    max = people[i];
            int maxsecond = 0;
            for(int i = 0; i < 10000; i++)
                if(maxsecond < people[i] & people[i] < max)
                    maxsecond = people[i];
            for(int i = 0; i < 10000; i++)
                if(maxsecond == people[i])
                    System.out.print(i+1 + " ");
            System.out.println();
        }
    }
}

입력을 받은 후 2등에 해당하는 점수를 구한 후 그 점수에 해당하는 사람들을 출력하도록 하였다.
