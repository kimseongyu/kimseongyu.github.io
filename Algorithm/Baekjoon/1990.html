1990번 소수인팰린드롬

  • Algorithm
  • Java
  • Date : 2020/07/21
  • https://www.acmicpc.net/problem/1990

문제요약
두 정수 a, b가 주어졌을 때, a이상 b이하인 소수인 팰린드롬을 모두 구하는 프로그램을 작성하시오. 

                            
public class Baekjoon1990 {
    public static void main(String[] args) throws Exception {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        StringTokenizer st = new StringTokenizer(br.readLine());
        
        int a = Integer.parseInt(st.nextToken());
        int b = Integer.parseInt(st.nextToken());
        boolean[] arr = new boolean[b+1];
        arr[1] = true;
        for (int i = 2; i * i <= b; i++) {
            if(arr[i])continue;
            for (int j = i * i; j <= b; j += i) {
                arr[j] = true;
            }
        }
        for(int i = a; i <= b; i++) {
            if(!arr[i] && func(Integer.toString(i))) {
                bw.write(i + "\n");;
            }
        }
        bw.write("-1");
        bw.flush();
        bw.close();
    }
    
    public static boolean func(String s) {
        boolean pal = true;
        int left = 0;
        int right = s.length()-1;
        while(left < right && pal) {
            if(s.charAt(left) != s.charAt(right)) {
                pal = false;
            }
            left++;
            right--;
        }
        return pal;
    }
}

에라토스테네스의 체를 사용하여 소수를 걸러내고 팰린드롬 검사를 하여 출력하였다.
