17298번 오큰수

  • Algorithm
  • Java
  • Date : 2020/07/22
  • https://www.acmicpc.net/problem/17298

문제요약
오큰수는 오른쪽에 있으면서 Ai보다 큰 수 중에서 가장 왼쪽에 있는 수를 의미한다. 그러한 수가 없는 경우에 오큰수는 -1이다.
크기가 N인 수열 A = A1, A2, ..., AN이 있다. 수열의 각 원소 Ai에 대해서 오큰수 NGE(i)를 구하시오. 

                            
public class Baekjoon17298 {
    public static void main(String[] args) throws Exception {
        BufferedReader br = new BufferedReader( new InputStreamReader( System.in ) );
        BufferedWriter bw = new BufferedWriter( new OutputStreamWriter( System.out ) );
        StringTokenizer st = new StringTokenizer( br.readLine() );
        
        int n = Integer.parseInt(st.nextToken());
        st = new StringTokenizer( br.readLine() );
        Stack in = new Stack<>();
        Stack s = new Stack<>();
        Stack an = new Stack<>();
        for(int i = 0; i < n; i++) {
            in.push(Integer.parseInt(st.nextToken()));
        }
        
        s.push(-1);
        for(int i = n-1; i >= 0; i--) {
            int a = in.pop();
            while(a >= s.peek() && s.peek() != -1) {
                s.pop();
            }
            an.push(s.peek());
            s.push(a);
        }
        
        for(int i = 0; i < n; i++) {
            bw.write(an.pop() + " ");
        }
        bw.flush();
        bw.close();
    }
}

오른쪽에서 왼쪽으로 가면서 자기보다 작은 수가 스택에 쌓여있으면 pop하도록 하였다.
