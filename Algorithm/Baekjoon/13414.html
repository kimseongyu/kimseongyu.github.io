13414번 수강신청

  • Algorithm
  • Java
  • Date : 2020/05/13
  • https://www.acmicpc.net/problem/13414

문제요약
수강신청 부하 관리 시스템을 도입하기로 결정하였다. 새로운 관리 시스템은 다음과 같은 방식으로 동작한다.
1. 수강신청 버튼이 활성화 된 후, 수강신청 버튼을 조금이라도 빨리 누른 학생이 대기목록에 먼저 들어간다.
2. 이미 대기열에 들어가 있는 상태에서 다시 수강신청 버튼을 누를 경우 대기목록의 맨 뒤로 밀려난다.
3. 잠시 후 수강신청 버튼이 비활성화 되면, 대기목록에서 가장 앞에 있는 학생부터 자동으로 수강신청이 완료되며, 수강 가능 인원이 꽉 찰 경우 나머지 대기목록은 무시하고 수강신청을 종료한다. 수강 가능 인원 k명과 대기목록의 길이 l과 l개의 학번이 주어질 때, 수강 신청한 학번을 출력하시오. 

                            
public class Baekjoon13414 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        int k = sc.nextInt();
        int l = sc.nextInt();
        Stack s = new Stack<>();
        for(int i = 0; i < l; i++)
            s.push(sc.next());
        
        LinkedHashSet set = new LinkedHashSet<>();
        for(int i = 1; i <= l; i++)
            set.add(s.pop());
        
        for(String c: set) {
            s.push(c);
        }
        
        for(int i = 1; i <= k; i++) {
            System.out.println(s.pop());
            if(s.empty())
                break;
        }
    }
}

Stack을 사용하여 입력받은 후 set으로 출력하여 중복을 없앤 뒤 다시 Stack에 삽입하고 입력된 순서대로 학번을 출력하였다.
