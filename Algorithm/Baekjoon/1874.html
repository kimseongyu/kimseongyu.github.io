1874번 스택 수열

  • Algorithm
  • Java
  • Date : 2020/07/30
  • https://www.acmicpc.net/problem/1874

문제요약
1부터 n까지의 수를 스택에 넣었다가 뽑아 늘어놓음으로써, 하나의 수열을 만들 수 있다. 임의의 수열이 주어졌을 때 스택을 이용해 그 수열을 만들 수 있는지 없는지, 있다면 어떤 순서로 push와 pop 연산을 수행해야 하는지를 알아낼 수 있다. 이를 계산하는 프로그램을 작성하라. 

                            
public class Baekjoon1874 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        int n = sc.nextInt();
        int num = 1;
        Stack st = new Stack<>();
        Queue an = new LinkedList<>();
        Queue qu = new LinkedList<>();
        for(int i = 0; i < n; i++) {
            qu.offer(sc.nextInt());
        }
        
        st.push(0);
        while(!qu.isEmpty()) {
            int i = qu.poll();
            for(; num <= n && st.peek() != i; num++) {
                st.push(num);
                an.offer('+');
            }
            if(st.peek() == i) {
                st.pop();
                an.offer('-');
            }
        }
        
        if(st.size() == 1) {
            while(!an.isEmpty()) {
                System.out.println(an.poll());
            }
        }else {
            System.out.print("NO");
        }
    }
}

Stack을 사용하여 배열에 수를 저장하고 Queue를 사용하여 배열의 순서와 답을 저장하였다.
