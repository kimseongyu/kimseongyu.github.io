6378번 디지털 루트

  • Algorithm
  • Date : 2020/01/21
  • https://www.acmicpc.net/problem/6378

문제요약
디지털 루트란 N을 이루고 있는 모든 자릿수를 더했을 때 한자리 숫자일 경우를 뜻한다.
양의 정수 N이 주어졌을 때, 그 수의 디지털 루트를 구하는 프로그램을 작성하시오.

                            
public class Baekjoon6378 {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        
        String a = input.next();
        while(!a.equals("0")) {
            System.out.println(sum(a));
            a = input.next();
        }
    }
    
    public static String sum(String s) {
        int num = 0;
        for(int i = 0; i < s.length(); i++) {
            num += s.charAt(i) - 48;
        }
        s = Integer.toString(num);
        if(s.length() == 1) {
            return s;
        }else {
            return sum(s);
        }
    }
}

문자열로 입력을 받아서 각 자릿수를 더하는 방식을 사용했다.
