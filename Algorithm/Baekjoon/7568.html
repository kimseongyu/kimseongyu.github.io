7568번 덩치

  • Algorithm
  • Java
  • Date : 2020/02/12
  • https://www.acmicpc.net/problem/7568

문제요약
키와 몸무게가 더 클 때 상대방보다 덩치가 더 크다고 말할 수 있다.
N명의 몸무게와 키가 담긴 입력을 읽어서 각 사람의 덩치 등수를 계산하여 출력하시오. 

                            
public class Baekjoon7568 {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        
        int a = input.nextInt();
        int[][] b = new int[a][3];
        for(int i = 0; i < a; i++) {
            b[i][0] = input.nextInt();
            b[i][1] = input.nextInt();
            b[i][2] = 1;
        }
        for(int i = 0; i < a; i++) {
            for(int j = 0; j < a; j++) {
                if(b[i][0] < b[j][0] & b[i][1] < b[j][1])
                    b[i][2]++;
            }
        }
        for(int i = 0; i < a; i++)
            System.out.print(b[i][2] + " ");
    }
}

2차원 배열을 활용하여 키와 몸무게 등수를 저장하여 브루트포싱을 통하여 각 값을 비교해 등수를 정했다.
