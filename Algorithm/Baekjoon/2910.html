2910번 빈도 정렬

  • Algorithm
  • Java
  • Date : 2020/05/07
  • https://www.acmicpc.net/problem/2910

문제요약
수열이 주어졌을 때, 빈도 정렬을 하는 프로그램을 작성하시오. 

                            
public class Baekjoon2910 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        int n = sc.nextInt();
        int c = sc.nextInt();
        ArrayList list1 = new ArrayList<>();
        ArrayList list2 = new ArrayList<>();
        for(int i = 0; i < n; i++) {
            int a = sc.nextInt();
            if(list1.contains(a)) {
                list2.set(list1.indexOf(a), list2.get(list1.indexOf(a))+1);
            }else {
                list1.add(a);
                list2.add(1);
            }
        }
        
        
        for(int i = 0; i < n; i++) {
            int max = 0;
            for(int j = 0; j < list2.size(); j++) {
                if(list2.get(j) > list2.get(max)) {
                    max = j;
                }
            }
            for(int j = 0; j < list2.get(max); j++) {
                System.out.print(list1.get(max) + " ");
            }
            list2.set(max, 0);
        }
    }
}

ArrayList를 이용하여 list1에는 입력된 순서대로 중복되지 않게 저장하고 list2에는 그 빈도를 저장하였다.
