10773번 제로

  • Algorithm
  • Java
  • Date : 2020/05/14
  • https://www.acmicpc.net/problem/10773

문제요약
잘못된 수를 부를 때마다 0을 입력하고, 가장 최근에 쓴 수를 지운다.
모든 수를 입력 받아 그 수의 합을 출력하시오. 

                            
public class Baekjoon10773 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        int k = sc.nextInt();
        Stack s = new Stack<>();
        for(int i = 0; i < k; i++) {
            int n = sc.nextInt();
            if(n == 0 && s.empty()) {
                continue;
            }else if(n == 0) {
                s.pop();
            }else {
                s.push(n);
            }
        }
        int add = 0;
        for(int i = 0; i < s.size(); i=i) {
            add += s.pop();
        }
        System.out.print(add);
    }
}

Stack을 사용하여 입력받은 후 0이 입력되면 pop을 하도록 하였다.
